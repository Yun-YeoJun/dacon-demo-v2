
  # Untitled

  This is a code bundle for Untitled. The original project is available at https://www.figma.com/design/2t1sEYchdkZLYcVDZTduQs/Untitled.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

## API 연결
- 기본: same-origin으로 `/api/v1/analyze` 호출
- 로컬 개발 시: `.env`에 `VITE_API_BASE=http://localhost:8000` 설정
